Interface VarDefn

  • All Known Implementing Classes:
    AbstractVarDefn, HeaderParam, InjectedParam, LocalVar, SoyDocParam, TemplateParam, UndeclaredVar

    public interface VarDefn
    Interface for the definition of a variable, i.e. a value that can be referred to by name. Variables include params, injected params, and local vars. Note that a definition does not always have a corresponding declaration in user code. Some definitions are implicit, e.g. in the case of injected params.
