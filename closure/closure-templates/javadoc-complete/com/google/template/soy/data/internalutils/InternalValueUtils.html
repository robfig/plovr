        Converts a compile-time globals map in user-provided format into one in the internal format.

        The returned map will have the same iteration order as the provided map.

        Parameters:
        compileTimeGlobalsMap - Map from compile-time global name to value. The values can be any of the Soy primitive types: null, boolean, integer, float (Java double), or string.
        Returns:
        An equivalent map in the internal format.
        Throws:
        SoySyntaxException - If the map contains an invalid value.
