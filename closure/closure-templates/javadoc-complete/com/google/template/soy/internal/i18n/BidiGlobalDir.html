com.google.template.soy.internal.i18n

Class BidiGlobalDir

  • java.lang.Object
    • com.google.template.soy.internal.i18n.BidiGlobalDir

  • public class BidiGlobalDir
extends Object
    Bidi global direction, which is either a "static" integer value (ltr=1, rtl=-1), or a code snippet yielding such a value when evaluated at template runtime.

    Important: Do not use outside of Soy code (treat as superpackage-private).

        public boolean isStaticValue()

      • getStaticValue

        public int getStaticValue()
        The "static" bidi global direction, as an integer: ltr=1, rtl=-1. If the bidi global direction is non-static, then calling this method will produce an exception.

      • getCodeSnippet

        public String getCodeSnippet()
        A source code snippet that evaluates at template runtime to the bidi global direction, i.e. one of the integer values 1 (ltr), 0 (unknown), and -1 (rtl). When the bidi global direction is static, returns a string representing an integer literal, e.g. "1". Thus, should never be null or empty.
