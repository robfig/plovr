com.google.template.soy.internal.targetexpr

Interface TargetExpr

  • All Known Implementing Classes:
    JsExpr

    public interface TargetExpr
    Value class to represent an expression in the target source (JS, Python, etc.). Includes the text of the expression as well as the precedence of the top-most operator.

    Important: Do not use outside of Soy code (treat as superpackage-private).
