Class BidiSpanWrapDirective

  • java.lang.Object
    • com.google.template.soy.bididirectives.BidiSpanWrapDirective
  • All Implemented Interfaces:
    SanitizedContentOperator, SoyJsSrcPrintDirective, SoyJavaPrintDirective, SoyPrintDirective

    public class BidiSpanWrapDirective
extends Object
implements SanitizedContentOperator, SoyJavaPrintDirective, SoyJsSrcPrintDirective
    A directive that maybe wraps the output within a 'span' with dir=ltr or dir=rtl. This wrapping is only applied when the output text's bidi directionality is different from the bidi global directionality.
