com.google.template.soy.shared.internal

Annotation Type ApiCallScope


  • @Target(value={TYPE,METHOD})
@Retention(value=RUNTIME)
public @interface ApiCallScope
    Guice scope for a Soy API method call. Only used in some API methods where it would be useful to be able to inject API call parameters (such as the message bundle) directly into lower-level objects (such as passes) created during the handling of the API call.

    Important: This may only be used in implementing plugins (e.g. functions, directives).
