    Serializable, Comparable<AutoescapeMode>

    public enum AutoescapeMode
extends Enum<AutoescapeMode>
    Specifies how the outputs of {print} commands that lack escaping directives are encoded.

    • Enum Constant Summary

      Enum Constants 
      Enum Constant and Description
      CONTEXTUAL
      Contextual auto-escaping is on for the template so directiveless prints will be escaped based on the surrounding context.
      FALSE
      Auto-escaping is off for the template.
      STRICT
      Strict form of contextual autoescaping in which no autoescape-cancelling print directives nor calls to non-strict templates are allowed.
      TRUE
      Auto-escaping is on for the template so directiveless prints will be HTML escaped.
