com.google.template.soy.soytree

Class ExprUnion

  • java.lang.Object
    • com.google.template.soy.soytree.ExprUnion

  • public class ExprUnion
extends Object
    Represents a Soy expression in either V2 or V1 syntax.

    Important: Do not use outside of Soy code (treat as superpackage-private). If this expression is in V2 syntax, then getExpr() will return a nonnull expression tree. If this expression is in V1 syntax, then #getExpr() will return null. In either case, the expression text can be obtained from getExprText().
