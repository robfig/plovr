    NodeVisitor<SoyNode,Void>

    public class ReportSyntaxVersionErrorsVisitor
extends AbstractSoyNodeVisitor<Void>
    Visitor for asserting that all the nodes in a parse tree or subtree conform to the user-declared syntax version.

    Important: Do not use outside of Soy code (treat as superpackage-private).

    exec(com.google.template.soy.soytree.SoyNode) may be called on any node. There is no return value. However, a SoySyntaxException is thrown if the given node or a descendant does not satisfy the user-declared syntax version.
