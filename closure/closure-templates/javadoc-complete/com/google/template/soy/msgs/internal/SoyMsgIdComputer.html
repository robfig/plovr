        Computes the unique message id for a message, given the message parts, the meaning string (if any), and the content type (if any). These are the only elements incorporated into the message id.

        In particular, note that the id of a message does not change when its desc changes.

        Parameters:
        msgParts - The parts of the message.
        meaning - The meaning string, or null if none (usually null).
        contentType - Content type of the document that this message will appear in (e.g. "text/html", or null if not used.
        Returns:
        The computed message id.
